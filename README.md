att-formatjson 
=====
att plugin - format your json file to readable


单元测试
------
[![travis build status](https://api.travis-ci.org/colorhook/att-formatjson.png)](https://www.travis-ci.org/colorhook/att-formatjson)

Usage
-------

```
att formatjson **/*.json
```

```
att formatjson **/*.json --indent 4
```
